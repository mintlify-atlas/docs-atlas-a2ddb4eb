## Overview

The `ViewToolApp` class (ViewToolApp.java:20) is the main JFrame window that serves as the application's primary container. It manages the overall UI structure, handles user authentication state, and controls navigation between different panels.

## Authentication Flow

### Initial State

On application startup, the window checks for a stored authentication token:

```java
token = tokenService.getToken();
if (token == null) {
    Login login = new Login(this);
    showPanel(login);
} else {
    loggedSuccess(token);
}
```

ViewToolApp.java:49-56

<Info>
If no token is found, the login panel is displayed. If a valid token exists, the user is automatically logged in and shown the main interface.
</Info>

### Successful Login

When authentication succeeds, the `loggedSuccess()` method:

1. Stores the authentication token
2. Sets logged in state to true
3. Shows the main view with management panel
4. Makes the settings button visible

```java
public final void loggedSuccess(String token) {
    this.token = token;
    this.loggedIn = true;
    showMainAndManagement(panelMain, panelManagement);
    buttonSetting.setVisible(true);
}
```

## Panel Management

### showPanel() Method

This method handles switching between single panels:

```java
private void showPanel(JPanel panel) {
    dinamicPanel.removeAll();
    panel.setOpaque(true);
    dinamicPanel.add(panel, BorderLayout.CENTER);
    
    if (!isLoggedIn()) {
        panelButtonMainAndSetting.setVisible(false);
    } else {
        panelButtonMainAndSetting.setVisible(true);
    }
    
    dinamicPanel.revalidate();
    dinamicPanel.repaint();
    pack();
    setLocationRelativeTo(null);
    
    buttonSetting.setVisible(false);
    buttonMain.setVisible(true);
}
```

ViewToolApp.java:138-159

### showMainAndManagement() Method

This special method displays two panels simultaneously:

```java
public void showMainAndManagement(JPanel top, JPanel bottom) {
    JPanel combined = new JPanel(new BorderLayout());
    top.setPreferredSize(new java.awt.Dimension(0, 167));
    combined.add(top, BorderLayout.NORTH);
    combined.add(bottom, BorderLayout.CENTER);
    showPanel(combined);
    buttonSetting.setVisible(true);
    buttonMain.setVisible(false);
    pack();
}
```

ViewToolApp.java:161-172

<Accordion title="Why combine panels?">
The main view combines PanelMain (download interface) and PanelManagement (library management) to provide a unified workflow where users can download content and immediately manage their media library without switching views.
</Accordion>

## Best Practices

<Tip>
When extending ViewToolApp:
- Always check `isLoggedIn()` before showing authenticated panels
- Use `showPanel()` for single views, `showMainAndManagement()` for combined views
- Call `pack()` and `setLocationRelativeTo(null)` to properly size and center the window
- Update navigation button visibility based on the current view
</Tip>
