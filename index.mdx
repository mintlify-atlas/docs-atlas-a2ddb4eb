## Quick Start

<Steps>
  <Step title="Install Requirements">
    Ensure you have Java 17+ and download yt-dlp and FFmpeg binaries
  </Step>
  <Step title="Configure Paths">
    Set up the paths to yt-dlp.exe and your preferred download directory
  </Step>
  <Step title="Start Downloading">
    Paste a YouTube URL, select your format, and start downloading
  </Step>
</Steps>

## Technical Highlights

- **Java Swing UI**: Modern desktop interface with retro aesthetic
- **yt-dlp Integration**: Powerful command-line downloader with full control
- **FFmpeg Support**: Audio extraction and format conversion
- **Network API**: REST integration with DI Media Network
- **Event-Driven**: Custom event system for real-time updates
- **Media Management**: Scan, filter, and organize local files

<Note>
  ViewTool is an educational project developed for the DAM (Desarrollo de Aplicaciones Multiplataforma) program, demonstrating interface design principles and Java desktop development.
</Note>
